# Ortak araçlar

Bu klasörde yalnız sınav biçiminden bağımsız yardımcılar bulunur:

- `check_scope.py`: program sahipliği dışındaki değişiklikleri reddeder.
- `manifest_merge.py`: ayrı SGS/SMMM kaynak manifestlerini çalışma zamanı
  manifestinde birleştirir.

Soru üretim mantığı, sınav kotaları veya mevzuat kararları ortak klasöre konmaz.
